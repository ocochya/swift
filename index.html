Swift

Swift is a new and fast growing progamming language created by Apple Inc.
It aims to allow users build iOS, macOS, watchOS, and tvOS apps. Swift is based on Objective-C, Rust, Haskell, Ruby, Python, C#, CLU, and many more existing languages combined with it's own modern and innovative features.
This makes it easy to pick up for new and also experienced programmers.
